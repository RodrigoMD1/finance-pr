### **🔍 Cómo usar el nuevo buscador:**

1. **Ve a la sección "Agregar Activos"**
2. **Usa el buscador inteligente** que aparece arriba del formulario
3. **Escribe cualquier cripto o acción:**
   - `Bitcoin` → Obtienes BTC con precio actual
   - `Apple` → Obtienes AAPL con precio actual 
   - `Tesla` → Obtienes TSLA con precio actual
   - `Cardano` → Obtienes ADA con precio actual
   - `Microsoft` → Obtienes MSFT con precio actual
